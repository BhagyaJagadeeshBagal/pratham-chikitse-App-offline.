# 🚑 Pratham-Chikitse – First Aid Emergency Offline Guide

Pratham-Chikitse is an Android-based emergency first-aid application developed using Generative AI technologies.  
The application provides medically guided first-aid instructions for emergency situations such as burns, choking, snake bites, fractures, bleeding, CPR, and more.

The major goal of this project is to provide quick and reliable first-aid guidance in offline mode, especially for people in rural or remote areas where internet access may not always be available.

# 📖 About the Project

During emergencies, people often panic and may follow incorrect home remedies due to lack of proper medical guidance.  
The first few minutes are critical in saving a person's life.

Pratham-Chikitse acts as a digital first-aid assistant that helps users perform immediate first-aid procedures through:
- Step-by-step guidance
- Offline emergency instructions
- AI-generated responses
- User-friendly Android interface
- Local language support

The app is designed to work efficiently even without internet connectivity.

---

# ❗ Problem Statement

In many emergency situations:
- People do not know proper first-aid procedures
- Wrong home remedies are followed
- Internet connectivity may not be available
- Medical help may take time to arrive

This creates the need for an offline emergency first-aid guidance application.

Pratham-Chikitse solves this problem by providing instant medically guided instructions directly on mobile devices.

- Select **Open Existing Project**

---

## 3️⃣ Sync Gradle Files

Allow Gradle synchronization to complete successfully.

---

## 4️⃣ Build the Project

Click:

```text
Build → Make Project
```

---

## 5️⃣ Run the Application

- Connect Android device or emulator
- Click the ▶ Run button
